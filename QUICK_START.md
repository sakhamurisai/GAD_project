## 🎯 **How to Start the Server**

### **Option 1: Using the Batch File (Easiest)**
1. Double-click `start-server.bat`
2. The server will start automatically
3. Open your browser to `http://localhost:3000`

### **Option 2: Using PowerShell Script**
1. Right-click `start-server.ps1`
2. Select "Run with PowerShell"
3. The server will start automatically
4. Open your browser to `http://localhost:3000`

### **Option 3: Manual Commands**
```powershell
# Set Node.js path
$env:PATH = "D:\Laptop related\softwares\nodejs;" + $env:PATH

# Navigate to project folder
cd D:\codes\gad_project

# Start the server
npm start
```

## 🔧 **Troubleshooting**

### **If Node.js is not found:**
1. Update the path in `start-server.bat` or `start-server.ps1`
2. Change `D:\Laptop related\softwares\nodejs` to your actual Node.js path

### **If port 3000 is busy:**
1. Stop any other applications using port 3000
2. Or change the port in `server.js` (line 12)

### **If dependencies are missing:**
1. Run `npm install` manually
2. Or use the provided scripts which handle this automatically
